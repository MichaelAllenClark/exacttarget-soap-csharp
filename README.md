# ExactTarget SOAP Starter Project for C# (.NET Framework 3.5 or greater)

This starter project contains some a working project showing how to access [ExactTarget][0]'s SOAP API using C# (.NET Framework 3.5 or greater)

## Prerequisites

### Getting Started 
Before making request, rename app.config.template to app.config  Open this file and input your username/password for your ExactTarget account in place of putUsernameHere and putPasswordHere.


### CODE@ ExactTarget's Developer Community Site

For more details on the ExactTarget Email SOAP API, please join our developer community at [CODE.ExactTarget.com][1]
